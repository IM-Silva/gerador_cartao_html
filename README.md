## 📌 Sobre o projeto

Este projeto em Python gera automaticamente um cartão personalizado em HTML com base nos dados inseridos pelo usuário.

O objetivo é praticar:
- Lógica de programação
- Manipulação de strings
- Geração dinâmica de HTML com Python
- Estrutura de projetos simples

---

## 🧠 Como funciona

O usuário informa no terminal:
- Nome
- Profissão
- Cor preferida (em inglês)

Depois disso, o Python gera um arquivo `index.html` com um cartão estilizado.
